## 🧩 项目说明

本仓库用于整理我在 HackTheBox 平台上攻克的 **Machines（靶机）** 的详细 Writeup。  
所有内容均为个人学习过程的真实记录，旨在：

- 巩固渗透测试技能
- 提升漏洞分析与利用能力
- 建立系统化的攻防知识体系
- 为其他 HTB 学习者提供参考

> ⚠️ 本仓库仅用于技术学习与交流，禁止用于任何非法用途。
